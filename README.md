# React Calculator

> **Status:** Frontend practice project

A calculator built in React to practice component composition, state updates, and predictable interaction logic.

## What This Repo Shows

- Focused on small but complete browser interactions instead of abstract exercises.
- Demonstrates forms, state handling, event flow, and simple UI decisions.
- Useful as a compact example of frontend fundamentals and implementation clarity.

## Tech

React, JavaScript, Create React App

## Run Locally

```bash
npm install
npm start
```

## Notes

This repo stays public as part of the path from small frontend projects to larger product work.
